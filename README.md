# cs2-map-bp-tool

该工具旨在为 CS2 玩家提供便捷的地图 banning（禁用）和 picking（选择）协作流程，简化比赛前的准备工作。

## 功能特点

- 房间创建与加入：支持创建房间并邀请他人加入，最多支持2名用户（两队队长）
- 队伍管理：
  - 自定义队伍名称
  - 队员选择与分配
  - 骰子投掷决定选人顺序和先后手
- 地图BP流程：
  - 自定义地图池
  - 完整的地图禁用（ban）和选择（pick）流程
  - 地图阵营选择与分配
- 实时协作：基于WebSocket的实时同步，确保双方操作一致
- 比赛信息汇总：自动生成包含队伍组成、地图选择和阵营分配的比赛摘要

## 安装与使用

### 前提条件

- Node.js (v14+)
- npm 或 yarn

### 安装步骤

1. 克隆本仓库
```bash
git clone https://github.com/Andycommander66/cs2-map-bp-tool/
cd cs2-map-bp-tool
```

2. 安装依赖
```bash
npm install
# 或
yarn install
```

3. 启动服务器
```bash
npm start
# 或
yarn start
```

4. 在浏览器中访问 `http://localhost:4900` 即可使用

## 使用流程

1. 输入昵称，创建房间或加入现有房间
2. 双方加入后，投掷骰子决定选人顺序和先后手
3. 按照决定的顺序选择队员
4. 进入地图BP阶段，按照流程进行地图禁用和选择
5. 完成后查看比赛信息汇总

## 界面说明

- 房间管理：创建或加入房间的入口
- 队员选择：分配队员和决定选人权的界面
- 地图BP：进行地图禁用和选择的主要界面
- 结算页面：显示最终确定的比赛信息
- 聊天窗口：实时沟通工具
